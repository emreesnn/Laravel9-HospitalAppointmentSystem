C) CSS Files and Structure - top

Styles

We're using 06 CSS files in this template. We have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

font-awesome.css

Base styles for Font Awesome icon fonts.

bootstrap.css

This Stylesheet contains responsive grid system, default element style based on bootstrap. It is bootstrap framework main css file

slick.css

Styles for slider (slick slider).

jquery.fancybox.css

It's Fancybox slider css file for gallery Popup view

default-theme.css

This is default theme color file for this template.

style.css

Common css style and structures for this template.
